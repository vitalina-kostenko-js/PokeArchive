## Naming

**Files** — use dot-notation suffixes:

| Type | Pattern |
|---|---|
| page | `page.tsx` |
| layout | `layout.tsx` |
| module | `name.module.tsx` |
| component | `name.component.tsx` |
| service | `name.service.ts` |
| store | `name.store.ts` |
| hook | `name.hook.tsx` |
| model | `name.model.ts` |
| api client | `name.api.ts` |
| query | `name.query.ts` |
| mutation | `name.mutation.ts` |
| constant | `name.constant.ts` |
| interface | `name.interface.ts` |

**Directories** — kebab-case. Singular for single components (`button`, `card`), plural for collections (`users`, `hooks`).
